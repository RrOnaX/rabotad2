# rabotaDavit2

Учебный проект, созданный через BAT-скрипт.

## Быстрый запуск
1. Откройте папку проекта в VS Code.
2. Запустите scripts\run_project.bat.
3. Для тестов используйте python -m pytest.

## Структура
src - исходный код
tests - тесты
docs - документация
scripts - вспомогательные скрипты
logs - логи запуска
reports - отчеты
screenshots - скриншоты работы
